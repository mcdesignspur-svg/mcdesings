01 — El problema

Antes de la automatización

Publicar un producto nuevo en una boutique de ropa requería varios pasos manuales repetitivos:

✗ Fotografiar el producto
✗ Escribir la descripción para Shopify — título, detalles, materiales, tallas
✗ Crear el caption para Instagram — con hashtags, emojis, call to action
✗ Subir todo manualmente a cada plataforma

Horas de trabajo repetitivo por cada producto. Con 10–20 piezas nuevas por semana, el tiempo se acumula rápido.

02 — El sistema

EL PIPELINE COMPLETO

6 nodos. 0 clics después de subir la foto.

📁
Paso 1 · Trigger Google Drive

Nueva foto detectada

El workflow se activa automáticamente cuando el cliente sube una foto a una carpeta específica en Google Drive. Sin botones, sin correos, sin mensajes.

⬇️
Paso 2 · Descarga Drive API

Descarga la imagen

El sistema descarga el archivo de imagen en binario para pasárselo directamente a Claude como contexto visual.

🤖
Paso 3 · Análisis IA Claude Vision

Claude analiza el producto

Claude recibe la imagen y genera dos outputs estructurados en un solo prompt: la descripción completa para Shopify y el caption listo para Instagram.

Prompt → "Eres un experto en marketing de moda para boutiques en Puerto Rico. Analiza esta imagen de producto y genera: DESCRIPCION_SHOPIFY: [título + descripción detallada en español] CAPTION_INSTAGRAM: [caption con emojis, hashtags, CTA]"
⚙️
Paso 4 · Procesamiento Code Node

Extrae el contenido

Un nodo de código parsea el output de Claude y separa la descripción de Shopify del caption de Instagram en campos distintos, listos para los pasos siguientes.

shopifyDescription: "Conjunto casual de verano en tela lino premium..." instagramCaption: "✨ New arrival! Este conjunto es todo lo que necesitas..."
🛍️
Paso 5 · Publicación Shopify API

Crea el borrador en Shopify

El sistema crea automáticamente un borrador de producto en Shopify con la descripción generada, la imagen adjunta y el caption de Instagram guardado como nota interna. El dueño solo revisa y da publish.

💬
Paso 6 · Notificación WhatsApp

Avisa al cliente por WhatsApp

El cliente recibe un mensaje de WhatsApp con el caption listo para pegar en Instagram y el link directo al borrador en Shopify para revisar y publicar.

WhatsApp → "✅ Nuevo producto listo para revisar. 📝 Caption IG: ✨ New arrival! Este conjunto es todo... 🛍️ Ver borrador en Shopify: https://beboutique.myshopify.com/admin/..."
