Claude Code Notion Prospecting

Cómo construí un sistema de prospecting con AI que encuentra clientes en PR solo

Miguel Cotto
Fundador, MC Designs

En MC Designs automatizo mi proceso de prospecting con Claude Code y Notion. En esta guía te explico exactamente cómo funciona el sistema — para que lo construyas tú mismo, o si prefieres, yo lo monto por ti.

No es un truco de ChatGPT. Es un agente de AI que busca negocios, verifica si tienen website, extrae su información de contacto, escribe un pitch personalizado por cada uno, y guarda todo en Notion — completamente solo. Puedes aprenderlo aquí o simplemente decirme qué industria quieres atacar y yo te entrego el sistema listo.

Las Herramientas

Para construir este motor automático solo hacen falta dos piezas de software modernas (si aún no tienes Claude Code instalado, puedes ver mi guía de configuración paso a paso aquí):

Claude Code

El CLI oficial de Anthropic. Una versión de Claude que vive en tu terminal y usa herramientas reales para interactuar con internet y APIs.

Costo: $20/m · claude.ai/download arrow_forward

Notion

Mi base de datos operante. Claude se conecta vía API para crear una entrada organizada por cada prospecto encontrado.

Costo: Gratis · notion.so arrow_forward

Algo importante a destacar: No estamos hablando de Claude.ai (la ventana de chat normal en en browser). Claude Code es un agente virtual que ejecuta acciones, no solo te responde preguntas pasivamente.

Las Capacidades del Agente

Para que el proceso funcione, el agente hace uso de tres herramientas (o capabilities) en cadena continua:

1. WebSearch

Busca en internet programáticamente. Para el prospecting corre queries parecidos a esto:

  • "restaurantes Santurce Puerto Rico"
  • "salones de belleza Condado PR contacto"

Él corre 2-3 búsquedas profundas por cada industria y área para conseguir el doble de candidatos de los que finalmente necesito.

2. WebFetch

Para cada negocio encontrado, Claude Code entra directamente a su página web y la evalúa como un humano:

  • ¿Existe y carga la página?
  • ¿Se ve moderno o desactualizado visualmente?
  • ¿Tiene buena adaptación en versión móvil?
  • ¿Hay un teléfono o email visible en el contacto?

Si la empresa no tiene website (algo común en PR), WebFetch raspa la información desde sus perfiles de Facebook o Instagram directamente.

3. Notion API

Una vez calificado cada lead, Claude Code manda el JSON y lo guarda automáticamente en una tabla de Notion. Cero copy/paste manual, cero abrir el app uno mismo.

El Flujo Completo, Paso a Paso

  1. 1

    Input Inicial

    Le digo qué buscar en el terminal: la industria, el área geográfica y la cantidad de clientes deseada.

  2. 2

    Minería de Candidatos

    WebSearch recolecta opciones en todo el internet (el doble de lo necesario para poder tener margen de filtrado).

  3. 3

    Evaluación Humana-Simulada

    WebFetch verifica uno por uno: el estado del website, la calidad del mismo y extrae la información de contacto.

  4. 4

    Scoring Automático

    Coloca labels de prioridad: 🔴 Sin web, 🟡 Desactualizado, 🟢 Oportunidad AI.

  5. 5

    Creación del Pitch

    Escribe personalmente 2-3 oraciones específicas para ese negocio basadas en las debilidades que encontró en la auditoría web.

  6. check

    Notion Push

    Inserta la fila en la base de datos de Notion.

El resultado: En 5 minutos tengo decenas de prospectos calificados, con toda la información de contacto y un borrador del pitch listo para enviarles. Horas de minería manual comprimidas a segundos.

Para mejores resultados: Crea un Lead Finder Skill

Puedes correr este proceso con prompts largos de manera manual cada vez que lo necesites. Pero para convertirlo verdaderamente en un "sistema" repetible — mismo flujo, mismo formato predecible, y el mismo output excelente cada vez — la mejor idea es tener un skill configurado dentro de Claude Code.

Un skill es un archivo liviano de instrucciones que le dice al agente autónomamente cómo ejecutar el proceso de principio a fin, bajo unos parámetros de negocio. Una vez que existe en tu computadora, lo invocas con un simple comando de conversación.

Cómo crearlo

La magia es que no tienes que ser un desarrollador o escribirlo tú a mano. Literalmente, le pides al mismo agente que programe el sistema para ti con este prompt base:

"Crea un lead-finder skill para [TU_NEGOCIO]. Quiero que busque [INDUSTRIA] en [TU_MERCADO/CIUDAD], verifique si estos negocios tienen website, consiga su información de contacto, les escriba un pitch personalizado como si fuese yo vendiendoles, y finalmente guarde toda esa data tabularmente en mi base de datos de Notion."

Luego de que lo mandas, el agente escribe su propio archivo, define el pipeline logico y te lo deja listo en bandeja de plata. En MC Designs empecé creando este pipeline exactamente de esta manera.

Cómo usar el skill en el día a día

Una vez completado el *setup*, prospectar es tan fácil como escribir en el chat:

> Encuentra 5 restaurantes en Santurce, Puerto Rico que no tengan website o lo tengan súper desactualizado.

Él detecta el llamado del skill particular y hace toda la magia por debajo del telón. Se pone a trabajar sin pedir nada a cambio.
