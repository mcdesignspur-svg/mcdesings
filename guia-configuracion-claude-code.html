Claude Code Terminal Setup

Cómo configurar Claude Code CLI en tu computadora

Miguel Cotto
Fundador, MC Designs

Claude Code es una herramienta que te permite usar Claude directamente desde la terminal de tu computadora. En vez de abrir un browser, escribes tus instrucciones en la terminal y Claude ejecuta tareas, edita archivos, busca información y hasta corre comandos — todo desde ahí. Esta guía te lleva paso a paso desde cero hasta tenerlo corriendo.

QUÉ NECESITAS ANTES DE EMPEZAR

  • Node.js — Es el motor que necesita Claude Code para correr. Descárgalo en: nodejs.org (instala la versión LTS).
  • Una cuenta de Anthropic — Créala gratis en: claude.ai
  • Terminal — En Mac viene incluida (búscala como "Terminal"). En Windows usa PowerShell o instala Windows Terminal.

PASO 1 — INSTALAR CLAUDE CODE

Abre tu terminal y corre este comando:

npm install -g @anthropic-ai/claude-code

Esto instala Claude Code globalmente en tu computadora. Solo tienes que hacerlo una vez. Para verificar que se instaló bien, corre:

claude --version

Si ves un número de versión, estás listo para el paso 2.

PASO 2 — INICIAR SESIÓN

Corre este comando en tu terminal:

claude

La primera vez, Claude Code abre automáticamente una ventana en tu browser para que inicies sesión con tu cuenta de Anthropic. Inicia sesión, autoriza el acceso y regresa a la terminal — listo, ya estás autenticado. No necesitas copiar ninguna key ni configurar nada manualmente.

PASO 3 — TU PRIMER COMANDO

Navega a cualquier carpeta de tu computadora y corre claude. Ahí puedes escribirle instrucciones directamente. Por ejemplo:

  • "Resume este archivo en 3 puntos"
  • "Edita este código para que haga X"
  • "Búscame todos los archivos que mencionen X"

Claude tiene acceso a los archivos en la carpeta donde lo abras — así puede leer, editar y crear archivos directamente.

OPCIONAL — ÚSALO DESDE VS CODE

Si no te gusta la terminal pura, puedes usar Claude Code directamente desde VS Code. Yo personalmente lo prefiero así — tienes todo en un solo lugar: tu código, tus archivos y Claude en el mismo panel.

Cómo instalarlo:

  • Descarga VS Code gratis en: code.visualstudio.com
  • Abre VS Code y ve al panel de extensiones (ícono de cuadraditos en el menú lateral, o Cmd+Shift+X)
  • Busca "Claude Code" en la barra de búsqueda
  • Haz clic en Install

Cómo usarlo:

Una vez instalada la extensión, Claude Code aparece como un panel dentro de VS Code. Abre cualquier carpeta de tu proyecto, haz clic en el panel de Claude y escríbele directo — igual que en la terminal, pero sin salir del editor. Se autentica con la misma cuenta que configuraste en el Paso 2, no necesitas hacer nada extra.

La ventaja: puedes ver tus archivos y hablar con Claude en la misma pantalla. Brutal para cuando estás trabajando en un proyecto y quieres que Claude edite código, revise archivos o te ayude a construir algo — todo sin cambiar de ventana.

CÓMO LO USO YO

Yo uso Claude Code como mi asistente ejecutivo. Desde VS Code manejo mi Content Calendar de Notion, genero ideas de contenido, busco leads, redacto propuestas y automatizo flujos con N8N — todo sin salir del editor.
